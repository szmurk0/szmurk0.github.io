      Komentarze:

      Komentarz blokowy: Jest zapisywany między /* */
      Komentarz jednowierszowy: Jest zapisywany //
      Komentarz jednowierszowy uniksowy: Jest zapisywany znakiem #

      Zmienne prederfiniowane to takie, które są super globalne. Przechowują informacje dotyczące konfiguracji bieżącego skryptu.

      Typy zmiennych:
      
        Typy skalarne, czyli proste:
          boolean,
          integer,
          float/double,
          string.

        Typy złożone:
          array,
          object.

        Typy specjalne:
          resource,
          null.
